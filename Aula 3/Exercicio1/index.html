Introdução

C++ (em português: lê-se "cê mais mais", em inglês lê-se see plus plus) é a linguagem de programação compilada multi-paradigma (seu suporte inclui linguagem imperativa, orientada a objetos e genérica) e de uso geral. Desde os anos 1990 é uma das linguagens comerciais mais populares, sendo bastante usada também na academia por seu grande desempenho e base de utilizadores. (Fonte Usada: SansSerif, tamanho 20pt)
